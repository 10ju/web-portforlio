Python Drawing App

Link to Drawing App

This drawing app was coded completely in python and allows you to make simple drawings. However the performance gets low after you draw for a while. You can press "b" to change background to light blue, "t" to change background to teal, and "w" to change the background back to white. Pressing "1" just erases the drawing.

Link to Paddle Game

This is a python "pong" game made in "Genius Hour". The game is extremely simple and has some bugs, but is completely functional for 2 players. Player 1 uses WASD as controls and Player 2 uses IJKL.
